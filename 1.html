HTML

Pergunta 1 - O que é HTML?

R: HTML é uma linguagem que possibilita a exibição de documentos de textos, imagens, entre outros nos sites.
Ela utiliza de diversas tags como: <p>(parágrafo), <li>(linha da lista), <h1>(titulo), <hr>(cria uma linha horizontal), entre outros

Pergunta 2 - Qual o Papel dele na Web?

  • Na web ele permite que o conteúdo da Web seja exibido;
  • Exibe textos, paragráfos, conteúdos das tabelas, imagens, botões, cria links, entre outros;
  • Ao exibir textos na Web, também permite que tenham-se marcações de negrito, itálico, sublinhado, etc.
