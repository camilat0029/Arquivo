Web e Internet

Diferença entre Internet e Web

A internet é uma rede de computadores e funciona utilizando a arquitetura (Cliente - Servidor), já a web funciona como uma
forma de gerenciar e compartilhar informações. Assim para a web funcionar ela precisa de internet, mas a internet não
precisa da necessariamente da web para funcionar, pois pode oferecer outros serviços além da web, como e-mail, ssh, entre outros
