Comandos GIT

  • git init- criar um repositório;
  • git add [file] :adiciona um arquivo no stage
  • git status : exibe o estado do repositório
  • git commit -m "msg" : cria um commit
  • git log : exibe os commits do repositório, da branch atual
  • git checkout referencia : Recupera o conteúdo da referencia para WK
  • git remote add nome url : definir um repositório remoto
  • git push url branch : envia a branch para o repositório remoto
  • git log --oneline
  • git reset referencia --hard
  • git checkout -b novabranch [referencia]
